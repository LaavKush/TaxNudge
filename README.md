## ✨ About the Project

> 💡 Finance should be simple, not stressful.

TaxNudge AI is an intelligent tax assistant designed for self-employed individuals in India.  
It helps users plan taxes daily, avoid last-minute stress, and make smarter financial decisions using AI.

- 🎯 Clean and minimal layout

## 🧠 Core Features

### 🤖 AI Tax Advisor  
Ask tax-related questions and get instant AI-powered answers  

### 📈 Income Prediction  
Predict annual income and estimated tax liability  

### 💡 AI ITR Tips  
Get personalized tax-saving tips based on profession  

### 🧬 Tax Personality  
Understand your financial habits and improve savings
